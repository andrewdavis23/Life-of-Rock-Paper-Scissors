## Rules
- Each agent moves randomly: up, left, down, or right
- If the two land in the same place, follow rock, paper, scissor rules to find which is deleted
- If two agents of the same category land in the same place, a third is created

Rock (Red)
Paper (Green)
Sciessors (Blue)
